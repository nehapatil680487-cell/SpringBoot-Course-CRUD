# SpringBoot Course CRUD Project

## Overview

This is a Spring Boot REST API project that performs CRUD (Create, Read, Update, Delete) operations on Course data.

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* Git & GitHub

## Features

* Add a new course
* View all courses
* View a course by ID
* Update course details
* Delete a course

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer

## API Endpoints

### Get All Courses

GET /courses

### Get Course By ID

GET /courses/{id}

### Add Course

POST /courses

### Update Course

PUT /courses

### Delete Course

DELETE /courses/{id}
